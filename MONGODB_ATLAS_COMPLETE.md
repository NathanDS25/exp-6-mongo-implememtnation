## 🚀 Getting Started Right Now

### Step 1: Visit MongoDB Atlas
```
https://www.mongodb.com/cloud/atlas
Click "Start free"
```

### Step 2: Create Account & Cluster
- Create account (takes 2 min)
- Create FREE cluster (takes 2 min)
- Wait for activation (takes 2-5 min)

### Step 3: Configure Backend
- Add database user (username: `contactmanager`)
- Whitelist your IP
- Get connection string
- Update `server/.env` with connection string

### Step 4: Run Backend
```bash
cd server
npm install  # If not done
npm start
```

### Step 5: Open Frontend
- Open `index-api.html` in browser
- Should see green status indicator
- Done! 🎉

## 🔑 Your Connection String Will Look Like

```
mongodb+srv://contactmanager:YOUR_PASSWORD@cluster0.xxxxx.mongodb.net/contact-manager?retryWrites=true&w=majority
```

Replace:
- `YOUR_PASSWORD` - The password you create
- `cluster0.xxxxx` - Your unique cluster path
